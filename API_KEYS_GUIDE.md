# API Keys Configuration Guide\n\nBu rehber sizə tətbiqinizdə API açarlarını necə konfiqurasiya etməyi göstərir.\n\n## 📋 Tələb olunan API Açarları\n\n### 1. Ödəniş Xidmətləri\n\n#### Stripe\n- **Nə üçün:** Kredit kartı ödənişləri\n- **Necə əldə etmək:** [Stripe Dashboard](https://dashboard.stripe.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `STRIPE_PUBLISHABLE_KEY`\n\n#### PayPal\n- **Nə üçün:** PayPal ödənişləri\n- **Necə əldə etmək:** [PayPal Developer](https://developer.paypal.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `PAYPAL_CLIENT_ID`\n\n### 2. Push Bildirişlər\n\n#### Expo Push Notifications\n- **Nə üçün:** Mobil push bildirişlər\n- **Necə əldə etmək:** Expo CLI ilə avtomatik yaradılır\n- **Konfiqurasiya:** `constants/config.ts` faylında `EXPO_PUSH_TOKEN`\n\n#### Firebase Cloud Messaging (FCM)\n- **Nə üçün:** Android push bildirişlər\n- **Necə əldə etmək:** [Firebase Console](https://console.firebase.google.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `FCM_SERVER_KEY`\n\n### 3. Analitika\n\n#### Google Analytics\n- **Nə üçün:** İstifadəçi davranışı analizi\n- **Necə əldə etmək:** [Google Analytics](https://analytics.google.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `GOOGLE_ANALYTICS_ID`\n\n#### Mixpanel\n- **Nə üçün:** Hadisə izləmə\n- **Necə əldə etmək:** [Mixpanel](https://mixpanel.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `MIXPANEL_TOKEN`\n\n### 4. Sosial Giriş\n\n#### Google OAuth\n- **Nə üçün:** Google ilə giriş\n- **Necə əldə etmək:** [Google Cloud Console](https://console.cloud.google.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `GOOGLE_CLIENT_ID`\n\n#### Facebook Login\n- **Nə üçün:** Facebook ilə giriş\n- **Necə əldə etmək:** [Facebook Developers](https://developers.facebook.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `FACEBOOK_APP_ID`\n\n### 5. Xəritə və Məkan\n\n#### Google Maps\n- **Nə üçün:** Xəritə və məkan xidmətləri\n- **Necə əldə etmək:** [Google Cloud Console](https://console.cloud.google.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `GOOGLE_MAPS_API_KEY`\n\n### 6. Fayl Saxlama\n\n#### Amazon S3\n- **Nə üçün:** Şəkil və fayl saxlama\n- **Necə əldə etmək:** [AWS Console](https://aws.amazon.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında:\n  - `AWS_ACCESS_KEY_ID`\n  - `AWS_SECRET_ACCESS_KEY`\n  - `AWS_REGION`\n  - `AWS_BUCKET_NAME`\n\n### 7. SMS Xidmətləri\n\n#### Twilio\n- **Nə üçün:** SMS göndərmə\n- **Necə əldə etmək:** [Twilio Console](https://console.twilio.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında:\n  - `TWILIO_ACCOUNT_SID`\n  - `TWILIO_AUTH_TOKEN`\n\n### 8. E-poçt Xidmətləri\n\n#### SendGrid\n- **Nə üçün:** E-poçt göndərmə\n- **Necə əldə etmək:** [SendGrid](https://sendgrid.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `SENDGRID_API_KEY`\n\n### 9. AI Xidmətləri\n\n#### OpenAI\n- **Nə üçün:** AI funksiyaları\n- **Necə əldə etmək:** [OpenAI Platform](https://platform.openai.com/)\n- **Konfiqurasiya:** `constants/config.ts` faylında `OPENAI_API_KEY`\n\n## 🔧 Konfiqurasiya Addımları\n\n### 1. Addım: API Açarlarını Əldə Edin\nYuxarıda qeyd olunan xidmətlərdən lazım olan API açarlarını əldə edin.\n\n### 2. Addım: Konfiqurasiya Faylını Yeniləyin\n`constants/config.ts` faylını açın və `your-api-key-here` yazılarını həqiqi API açarlarınızla əvəz edin:\n\n```typescript\nexport const API_CONFIG = {\n  // Məsələn:\n  STRIPE_PUBLISHABLE_KEY: 'pk_test_51ABC123...', // Həqiqi Stripe açarınız\n  GOOGLE_MAPS_API_KEY: 'AIzaSyABC123...', // Həqiqi Google Maps açarınız\n  // ... digər açarlar\n};\n```\n\n### 3. Addım: Backend URL-ni Təyin Edin\n`BASE_URL`-i öz backend serverinizin URL-i ilə əvəz edin:\n\n```typescript\nBASE_URL: Platform.select({\n  web: 'https://your-api-domain.com/api',\n  default: 'https://your-api-domain.com/api'\n}),\n```\n\n### 4. Addım: Xidmətləri Test Edin\nTətbiqi işə salın və konsol loglarında xidmətlərin düzgün işlədiyini yoxlayın:\n\n```\n✓ Auth service initialized\n✓ Analytics service initialized\n✓ API service configured with auth token\n✓ All services initialized successfully\n```\n\n## 🔒 Təhlükəsizlik Qeydləri\n\n1. **Heç vaxt** API açarlarını kod repozitoriyasında saxlamayın\n2. Environment variables istifadə edin production üçün\n3. API açarlarını yalnız lazım olan icazələrlə məhdudlaşdırın\n4. Müntəzəm olaraq API açarlarını yeniləyin\n5.
